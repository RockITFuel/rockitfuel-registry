# shadcn/ui Chart Variants Reference

A comprehensive reference of all chart variants available from [shadcn/ui](https://ui.shadcn.com/charts). These charts are built with Recharts.

| Area Chart - Legend | With legend |
| Area Chart - Stacked | Stacked areas |
| Area Chart - Stacked Expanded | Expanded stacked (100%) |
| Area Chart - Icons | With icons in legend |
| Area Chart - Gradient | With gradient fill |
| Area Chart - Axes | With custom axes |
